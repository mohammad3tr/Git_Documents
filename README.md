# Git_Documents
Document about Git learning


گیت، یک سیستم کنترل نسخه منبع آزاد و رایگان است که در ابتدا Linus Torvalds  در سال 2005 ایجاد کرد.
</br>
گیت توزیع می شود: به این معنی که هر توسعه دهنده، تاریخچه ی کامل مخزن کد خود را به صورت محلی دارد. این باعث می شود که Clone اولیه مخزن کندتر شود، اما عملیات بعدی مانند Commit, blame, diff, Merge و Log سریع تر انجام می شود. </br>
وی‌سی‌اس (سیستم کنترل نسخه) Version Control System یا به اختصار VCS به ابزاری اطلاق می‌گردد که این امکان را در اختیارمان می‌گذارد تا دست به مدیریت نسخه‌‌بندی یک پروژه بزنیم.</br>
